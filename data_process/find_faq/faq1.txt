🔗 10个追问链主题（每链2-4轮逻辑递进）
误食巧克力
→ 家庭能否催吐？→ 兽医指导下的规范操作要点 → 送医需携带的关键信息
中暑急救
→ 冰水降温风险 → 昏迷犬禁水原因 → 安全转运体位与监测指标
骨折处理
→ 自行复位危害 → 临时固定原则（材料/手法） → 送医前禁食必要性
误食异物
→ “促排偏方”风险 → 观察期核心监测指标（精神/排便/腹痛） → 手术指征判断
母犬分娩
→ 人工助产禁忌 → 新生幼犬拒食的规范处理 → 产后异常症状识别（恶露/体温）
癫痫发作
→ 按压/塞物危害 → 单次短时发作处理流程 → 发作记录要点（视频/时长/诱因）
蜂蜇过敏
→ 毒刺规范移除手法 → 过敏反应分级识别（轻度肿胀 vs 呼吸困难） → 急救药物使用前提
急性呕吐
→ 呕吐期喂食风险 → 呕吐物样本保留规范 → 需紧急送医的“红旗症状”
攻击行为纠正
→ 体罚后果（恐惧性攻击） → 攻击前兆识别（僵直/低吼） → 正向脱敏训练原则
皮肤病用药
→ 人用药膏毒性机制 → 初步自查要点（分布/分泌物/季节性） → 就医前信息准备清单
